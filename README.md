# anti-ai-writing

去除中文 AI 写作痕迹的 skill，51类模式，9语体门检体系。

## 功能

- 识别并去除 AI 生成文字的痕迹
- 支持9种语体：社交/口语、内容/自媒体、商务/职场、书面/一般、叙事非虚构/特稿、品牌广告/文案、学术/科技、公文/法律、高考/应试作文
- 强制门检：先判断是否真人文本，避免误改真人声口
- 终稿附打磨报告，清楚显示动了什么

## 使用方式

在支持 Hermes skill 的 AI 产品中直接调用：

```
/anti-ai-writing
```

或将文本贴给 AI，说"帮我去AI味"即可自动触发。

## 适用场景

玄仁黄历、公众号、抖音口播、小红书文案、博客等中文内容生产。

## 遵循规范

参照 Apple 大中华区、Nike 大中华区品牌文案作为品牌广告语体的反向诊断基准。
